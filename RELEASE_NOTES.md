# Anime Vault — Release Notes

## v2.0.0 (build 3) — 2026-05-13
**Major update — submitted to Google Play production**

### New Features
- Search wallpapers by character or series name
- Category filter chips (DBZ, Naruto, One Piece, Demon Slayer, JJK, Chainsaw Man, AOT, Bleach, OPM, HxH, Solo Leveling)
- Full-screen lock screen & home screen preview pages
- Set wallpaper directly to home screen, lock screen, or both
- Press/hover visual feedback on Set Wallpaper options (gold border + tint)

### Content
- 32 HD wallpapers across 11 iconic anime series
- Dragon Ball Z: Goku, Vegeta, Broly
- Naruto: Naruto, Sasuke, Kakashi, Itachi
- One Piece: Luffy (x3), Zoro, Sanji, Nami
- Demon Slayer: Tanjiro, Rengoku, Shinobu
- Jujutsu Kaisen: Gojo, Sukuna, Tokyo Squad
- Chainsaw Man: Denji (x2), Aki
- Attack on Titan: Levi
- Bleach: Ichigo
- One Punch Man: Saitama, Genos
- Hunter x Hunter: Killua
- Solo Leveling: Sung Jin-Woo

### UI / Branding
- Renamed app to "Anime Vault" (gold + crown header)
- Refreshed dark UI with gold (#FFB703) accent theme
